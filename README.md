## <h2> 📝 Description <h2>

<b>Laman.ai</b> is a high-performance web application designed to transform raw product or service descriptions into structured, high-converting sales pages in seconds. Developed as part of a technical assessment for <b>PT Dakwah Digital Network</b>, this tool handles professional copywriting and elegant layout design instantly.

<br />

## <h2> ✨ Key Features <h2>

<table width="100%">
  <tr>
    <td width="50%">
      <b>🧠 AI Copywriting Engine</b><br />
      Powered by <b>Groq SDK (Llama-3.3-70b-versatile)</b> to generate persuasive headlines, features, and benefits in natural Indonesian.
    </td>
    <td width="50%">
      <b>🎨 Dynamic Styling</b><br />
      Supports multiple aesthetics including <i>Glassmorphism, Neo-Brutalism, Dark Tech, and Minimalist</i>.
    </td>
  </tr>
  <tr>
    <td>
      <b>👁️ Live Preview</b><br />
      Instant responsive preview of the generated landing page layout.
    </td>
    <td>
      <b>💬 Smart Refinement</b><br />
      Refine specific sections using natural language feedback (e.g., "make the text bolder" or "change the CTA").
    </td>
  </tr>
  <tr>
    <td>
      <b>🔐 Auth & Persistence</b><br />
      Secure user authentication and project history management using <b>Supabase</b>.
    </td>
    <td>
      <b>📄 Code Export</b><br />
      Export generated pages into clean, semantic HTML and Tailwind CSS code.
    </td>
  </tr>
</table>
